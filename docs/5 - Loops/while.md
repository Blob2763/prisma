# `while`
Repeats code while a condition is met

## Usage
Output the numbers 1-5 in order:
```
set i = 1;
while (i <= 5) {
	output(i);
	set i = i + 1;
}
```
### Output
```
1
2
3
4
5
```

## Parameters
`while` expects exactly 1 parameter.
| Parameter | Type | Meaning |
|-|-|-|
| `condition` | Boolean | The condition that has to be met for the code to run |
